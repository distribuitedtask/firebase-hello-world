¡Hola desde Firebase Hosting!

Esta aplicación Hello World forma parte del entregable para la rúbrica de proveedores cloud. El despliegue se realiza mediante Firebase Hosting y se complementa con un contenedor Docker que permite validar la compilación local.

Comienza ejecutando firebase emulators:start para probar en local o firebase deploy para publicar esta página.
